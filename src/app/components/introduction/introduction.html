1.1. Введение в CSS

CSS (Cascading Style Sheets) — это язык стилей, который отвечает за внешний вид веб-страниц: цвета, шрифты, расположение элементов и адаптивность.

Анатомия CSS-правила

CSS состоит из правил. Каждое правило указывает браузеру, к какому элементу применить стили и какие именно свойства изменить.

.box {{ '{' }} background-color: blue; {{ '}' }}

Селектор (.box) — указывает, какой элемент (или элементы) на странице нужно стилизовать.

Свойство (background-color) — какую именно характеристику элемента мы хотим изменить.

Значение (blue) — новое значение для этого свойства.

Способы подключения CSS

Существует три способа добавить стили на страницу. Нажми на вкладки, чтобы увидеть синтаксис для каждого метода:

@if (linkingMethod() === 'inline') {

Стили пишутся прямо в HTML-теге через атрибут style. Метод применяется только к одному конкретному элементу на странице.

<div style="background-color: {{ selectedColor() }}; border-radius: {{ selectedRadius() }}; padding: {{ selectedPadding() }};">CSS</div>
} @if (linkingMethod() === 'internal') {

Стили пишутся внутри HTML-файла в специальном теге <style> (обычно внутри тега <head>).

<style>
  .box {{ '{' }}
    background-color: {{ selectedColor() }};
    border-radius: {{ selectedRadius() }};
    padding: {{ selectedPadding() }};
  {{ '}' }}
</style>
} @if (linkingMethod() === 'external') {

Самый лучший и рекомендуемый способ. Стили пишутся в отдельном файле с расширением .css и подключаются в HTML через тег <link>.

<!-- В HTML-файле внутри <head> -->
<link rel="stylesheet" href="styles.css">

<!-- В файле styles.css -->
.box {{ '{' }}
  background-color: {{ selectedColor() }};
  border-radius: {{ selectedRadius() }};
  padding: {{ selectedPadding() }};
{{ '}' }}
}
