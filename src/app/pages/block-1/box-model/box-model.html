1.3. Блочная модель (Box Model)

Каждый элемент в CSS представляется в виде прямоугольного бокса, состоящего из четырех слоев: контента, внутренних отступов (padding), рамки (border) и внешних полей (margin).

Понимание Box-Sizing

По умолчанию браузеры используют значение content-box. Это означает, что свойства width и height задают габариты только для области контента, а рамки (border) и внутренние отступы (padding) приплюсовываются сверху, физически увеличивая итоговый размер элемента на странице.

Свойство box-sizing: border-box заставляет браузер включать внутренние отступы и рамки в общую ширину и высоту. Если элементу задана ширина 200px, он останется ровно 200px, а размер внутренней области автоматически ужмется.

В профессиональной верстке хорошим тоном является переопределение box-sizing для всех элементов на странице с помощью универсального селектора:

*, *::before, *::after {{ '{' }}
  box-sizing: border-box;
{{ '}' }}

Вертикальные внешние отступы (margin-top и margin-bottom) двух соседних блоков в обычном потоке документа могут схлопываться в один общий отступ. Вместо суммы двух полей применится только один — максимальный из них.
